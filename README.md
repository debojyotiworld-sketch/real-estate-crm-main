## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/phoenixitservices/phoenix-realesthatic-crm.git
```

### 2. Go To Project Directory

```bash
cd phoenix-realesthatic-crm
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 5. Start Development Server

```bash
npm run dev
```
